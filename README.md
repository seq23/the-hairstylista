# The Hairstylista Site

Level 1 static marketing website for Nia Nowlin, The Hairstylista in Houston, TX.

## Scope

- Static HTML/CSS/JS only
- No environment variables
- No database
- No auth
- No contact form backend
- Cloudflare Pages compatible

## Commands

```bash
npm run validate
npm run build
npm run release:prepush:container
```

## Deployment

Cloudflare Pages build command:

```bash
npm run build
```

Cloudflare Pages output directory:

```bash
dist
```

## Brand

- Blush / gold / cream
- Feminine typography
- Flirty, fun, soft-glam tone
- Light animated flash words in the hero
- Appointment-only booking CTAs
