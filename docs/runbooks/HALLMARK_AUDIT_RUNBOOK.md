# Hallmark Audit Runbook — The Hairstylista Site

## Applicability

This is a Level 1 public static marketing website. Full local Hallmark browser evidence is optional before launch unless the operator wants a formal visual audit.

## Required Visual Surfaces

- `/` desktop
- `/` mobile
- Hero section
- Services section
- Signature U-part install section
- Gallery section
- Contact section

## Known Brand Constraints

- Feminine fonts
- Blush / gold / cream palette
- Flirty and fun tone
- Light animated flash words
- No backend contact form
